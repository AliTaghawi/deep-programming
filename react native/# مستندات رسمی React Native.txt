# مستندات رسمی React Native

برای یادگیری React Native از رفرنس‌های اصلی و معتبر استفاده کن تا همیشه به‌روز باشی و با بهترین شیوه‌ها آشنا شوی.

---

## مسیر یادگیری در مستندات

1. شروع سریع (Getting Started)  
   - انتخاب Expo CLI یا React Native CLI  
   - راه‌اندازی پروژه جدید و اجرای اولین اپ  
2. آموزش گام به گام (Tutorial: Building Projects)  
   - ساخت اولین کامپوننت  
   - مدیریت State و Props  
3. یادگیری اصول پایه (Learn the Basics)  
   - View, Text, Image, ScrollView  
   - استایل‌دهی با Flexbox  
   - Hooks مثل useState و useEffect  
4. راهنماهای تخصصی (Guides)  
   - ناوبری با React Navigation  
   - دسترسی به سخت‌افزار و Native Modules  
   - بهینه‌سازی عملکرد و فعال‌سازی Hermes  
5. مستندات API (API Reference)  
   - تمام کامپوننت‌ها و ماژول‌های داخلی  
   - مثال‌های کاربردی برای هر متد
